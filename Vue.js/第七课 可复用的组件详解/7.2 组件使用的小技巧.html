技巧1：推荐使用小写字母 + '-' 的方式进行组件命名。(也可以只是小写字母)
技巧2：template中的内容必须被一个/多个DOM元素包裹。(div元素/span元素 ...)
技巧3：在组件中，可以使用 template, data, methods, computed等多个选项。(和Vue实例类似)
技巧4：组件中的 data 必须是一个方法：data:function(){} / data(){} , 不能是对象：data:{}
Vue实例中的 data 可以是对象 / 方法


Bug:两个按钮同时被触发,因为两个按钮共用了一个 count
组件实现：每次点击只触发一个按钮, 每个按钮都有一个独属于自己的 count
